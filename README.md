# Front Library

@version: 6.0.11


## Use

Just import features as usual:

```js
import { extend } from 'front-library/Helpers/Extend';
```


## Modernizr addons

They are made to be concatenated at the end of the Modernizr file.

```js
[
    "modernizr.js",
    "node_modules/front-library/ModernizrAddons/mobile.js"
]
```
